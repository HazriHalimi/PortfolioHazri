# Hazri Halimi — Portfolio

Premium personal portfolio website for Hazri Halimi Izhar — Final-Year Computer Science Student at Universiti Malaysia Terengganu.

This repository contains a modern, single-page portfolio built with:

- HTML5, CSS3 (Glassmorphism, Aurora gradients), Bootstrap 5
- JavaScript (Typed.js, GSAP, Vanilla Tilt, AOS)
- Font: Poppins

Sections included:
- Hero
- About
- Skills
- Featured Projects (HydroAlert featured)
- Experience
- Education
- Pakistan Mobility
- GitHub
- Resume
- Contact

Preview locally:

```powershell
cd "c:\Users\hazzr\OneDrive\Documents\NetBeansProjects\HazriPortfolio"
py -m http.server 8000
# then open http://localhost:8000/public_html/index.html
```
